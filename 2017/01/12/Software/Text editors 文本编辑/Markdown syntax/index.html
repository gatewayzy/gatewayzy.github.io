Markdown syntax

发表于 2017-01-12   |   分类于 Softwares

说明：Markdown语法简单、面向纯文本、文本与渲染分离，强大而通用。介绍markdown语法，以及一些md编辑器。

参考文章：

Markdown简介

  • markdown好处在于较为通用的文本样式，如多级列表、多级标题等非常简单，有Github等网站的大力支持。
  • 缺点在于表格书写麻烦、图片只能以链接引入，所以图片只能先上传到公网，如免费的图片服务网站imgur。
  • 本文markdown简写为md。

图片链接

图片链接也支持直接式与参考式链接。直接式会以感叹号开始，表示是一个图片。

参考式也和参考式链接类似，用一个链接的方式定义实际地址，引用的时候使用标识符或者链接文本，并且实际地址的定义在文中无要求。
